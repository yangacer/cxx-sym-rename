```
OVERVIEW:
  Rename symbols per line from stdin to stdout
USAGE:
  ./cxx-sym-rename [--help] [--prefix=<string>] [--salt=<string>]
OPTIONS:
  --help              Print this message.
  --prefix=<string>   Prepend <string> to all renamed symbols.
  --salt=<string>     Add <string> as salt of renaming encryption.
```
